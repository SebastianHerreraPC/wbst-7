Transformaciones con css

Las transformaciones con css tienen cuatro estados:

Translate o desplazar, mueven el elemento de un lugar a otro

Scale o escalar, sirve para cambiar el tamaño de el elemento

Rotate o rotar, rota el elemento tal cual

Skew o estirar, estira al elemento hacia donde se indique

Las transiciones tiene 4 subpropiedades

  • transition-property (Que es lo que se cambia)
  • transition-duration (Tiempo)
  • transition-timing-function (Velocidad de animación)
  • transition-delay (Cuantos segundos tardara )
